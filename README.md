## Exercise 3.9: DBaaS vs DIY Comparison
- **DBaaS (Google Cloud SQL):** Automated backups and high availability, but higher cost.
- **DIY (Postgres in Container):** Full control and lower cost, but requires manual backup management (see 3.10).
