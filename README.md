## Exercise 3.9: DBaaS vs DIY Comparison
- **DBaaS (Google Cloud SQL):** Pros include automated backups, high availability, and minimal maintenance. Cons: Higher cost.
- **DIY (Postgres in Container):** Pros are full control and lower cost. Cons: Requires manual backup setup and high maintenance effort.
